# Amazon Clone

A simple Amazon homepage clone built using **HTML** and **CSS**. This project recreates the basic layout and design of Amazon's landing page for learning and practice purposes.

## Features

- Hero section
- Product category cards
- Clean and modern UI
- Footer similar to Amazon

## Technologies Used

- HTML5
- CSS3
